hello world!
this project pull data from an API - https://randomuser.me/ then, create different datasets and upload it to a remote database. Using PyCharm and different libraries like requests and pandas.
Prerequisite Installation
1.	project packages requirement:
    a.install MySQL-connector-python
    b.install PyMySQL
2.	fill in the config template for the database connector
3.	fill in the Api_conection -query_params:	
    a.results=how many users you want to fetch 
    b.seed=type here any word that you want if you want to get the same data every time.
    c.For more information visit https://randomuser.me/documentation
